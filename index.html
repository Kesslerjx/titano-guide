Titano

Guide

Required

  • MetaMask (wallet that supports BSC)

Setup

  • Add BSC to MetaMask
    1. Open MetaMask
    2. Go to Settings
    3. Got to Networks
    4. Press Add Network
      • Network Name: Smart Chain
      • https://bsc-dataseed.binance.org/
      • 56
      • BNB
      • https://bscscan.com
    5. Press Save
  • Add Titano to MetaMask
    1. Visit CoinMarketCap
    2. Copy the contract
    3. Open MetaMask (ensure you're in the Smart Chain wallet and not Ethereum)
    4. Press Import Tokens
    5. Paste the contract in the Token Address field (everything else should autofill)
    6. Press Import
